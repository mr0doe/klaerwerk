■ About

Klaerwerk is a personal dev playground started in early 2010. No investors, no hype—just coffee-fueled late-night coding sessions. Tools here are for private use first, then released for the curious. If it breaks your browser in 2010, congrats.

■ Core Features

  • Easy single-file distribution
  • Minimal dependencies—works in IE8+
  • Offline-ready HTML tools
  • Shell scripts for UNIX devotees
  • Encrypted redirects via GitHub API
  • Mock consoles and dev-themed UIs

■ Installation

No package manager? No problem:

  git clone https://github.com/storage4me/klaerwerk.git
  cd klaerwerk
  # pick your tool, open directly in browser or terminal

All HTML files run locally; all Fish/Python scripts just chmod +x and execute.

■ Usage Examples

a) netlogx.py

  # monitor network (no logs)
  python3 netlogx.py --status

b) bf.fish

  # search files, show ASCII table
  ./bf.fish 0 report.pdf /home/user/docs

c) secure-redirect.html

Open in browser, drop link, get encrypted GitHub redirect URL.
